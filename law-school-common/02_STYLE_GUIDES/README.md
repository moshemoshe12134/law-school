# 02_STYLE_GUIDES - Writing Standards

**Purpose:** Contains all writing standards, style conventions, and formatting guidelines for law school documents.

## Structure

### Universal Writing Guides
- `WRITING_FOUNDATIONS.md` - Universal writing principles (Elements of Style + AI guardrails + Legal writing conventions)

### Course-Specific Style Guides
Course-specific guides are created in each course's `00_ADMIN/` folder:
- `CrimLaw/00_ADMIN/course_style_guide.md`
- `Property/00_ADMIN/course_style_guide.md`
- `Torts/00_ADMIN/course_style_guide.md`
- `Deals/00_ADMIN/course_style_guide.md`

Each course guide includes:
- Course priorities (MPC focus, policy weight, case depth)
- Professor emphasis patterns
- Prep doc format preferences
- Exam answer style
- Case brief format (standardized structure from _PREP_FORMAT_EXAMPLE.md)

### Reference Materials
- `REFERENCES/elements-of-style.md` - Strunk & White writing principles (original reference)
- `REFERENCES/_PREP_FORMAT_EXAMPLE.md` - User's refined case brief format (DO NOT DELETE)
- `REFERENCES/echo360_transcript_guide.md` - Transcript editing guide (moved from root)

## Usage

1. **When generating any document:** Apply standards from `WRITING_FOUNDATIONS.md`
2. **When generating course-specific content:** Apply course-specific standards from `00_ADMIN/course_style_guide.md`
3. **When writing case briefs:** Follow format in `REFERENCES/_PREP_FORMAT_EXAMPLE.md`
4. **When editing transcripts:** Reference `echo360_transcript_guide.md`

## Style Hierarchy

1. **WRITING_FOUNDATIONS.md** - Universal baseline for all documents
2. **course_style_guide.md** - Course-specific modifications and additions
3. Course-specific guides trump universal guide where they differ (by design)

## Principles

- Clear, concise, active voice writing
- IRAC structure for legal analysis
- Consistent formatting across all documents
- AI guardrails prevent quality drift
