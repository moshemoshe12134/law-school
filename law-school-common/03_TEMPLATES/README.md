# 03_TEMPLATES - Standard Templates

**Purpose:** Contains all standard templates for generating course documents.

## Template Categories

### Preparation Templates
- `audio_prep_template.md` - Template for audio preps (TTS-friendly, conversational)
- `text_prep_template.md` - Template for text preps (searchable Q&A format)

### Review Templates
- `review_template.md` - Template for post-class review documents (with source extraction, prewrite identification)
- `review_index_template.md` - Template for tracking transcripts and review progress

### Outline & Sources Templates
- `sources_table_template.md` - Running table of all sources discussed in class

### Administrative Templates
- `master_log_template.md` - Template for course MASTER_LOG.md
- `exam_spec_template.md` - Template for exam specification documents
- `course_style_guide_template.md` - Template for course-specific style guides

### Prewrite Templates
- `prewrite_doctrinal_template.md` - Template for IRAC doctrinal analyses
- `prewrite_policy_template.md` - Template for policy argument documents

### Quality Control Templates
- `qc_checklist_template.md` - Quality control standards checklist

## Usage

1. **When creating a new course:** Copy `master_log_template.md` and `course_style_guide_template.md`
2. **When preparing for class:** Use `audio_prep_template.md` and `text_prep_template.md`
3. **When processing transcripts:** Use `review_index_template.md` to create tracking, then `review_template.md` for each class
4. **After reviews:** Use `sources_table_template.md` to track all sources mentioned in class
5. **Building exam materials:** Use `prewrite_doctrinal_template.md` and `prewrite_policy_template.md`
6. **Quality control:** Use `qc_checklist_template.md` to review any document

## Template Principles

- All templates are starting points (modify as needed for specific courses)
- Templates incorporate style standards from `02_STYLE_GUIDES/`
- Templates are designed to work with workflows in `01_WORKFLOWS/`
- Templates evolve based on what works (update them after discovering improvements)

## Standard Fields

All templates include:
- Clear document title
- Purpose statement
- Section headers (standardized)
- Formatting instructions
- Example content (where helpful)
- Style guide references

## Template Maintenance

- Update templates when you discover better structures
- Keep templates backward-compatible when possible
- Document changes in template header (date + what changed)
