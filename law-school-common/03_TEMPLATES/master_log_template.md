## QC Legend

**QC Iteration Count:**
- `0` = Not yet reviewed
- `1` = First QC pass complete
- `2+` = Additional QC passes completed
- `✓` = Final/passed QC

**Process:**
- QC happens by editing the document in place (no separate QC docs)
- Each QC pass increments the counter
- QC findings go in Notes column or in doc frontmatter
- Mark `✓` when document is exam-ready
