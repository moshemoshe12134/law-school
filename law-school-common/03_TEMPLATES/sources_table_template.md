## Usage Instructions

### Adding Sources After Each Review

1. Open the class review document
2. Extract all sources mentioned (cases, statutes, readings, scholars)
3. For each source:
   - Check if already in this table
   - If new: add complete entry
   - If existing: update "Classes Referenced" column
4. Update the "Last Updated" date

### Exam Relevance Scale

- ⭐ = Mentioned but not emphasized
- ⭐⭐ = Professor spent time on it; medium importance
- ⭐⭐⭐ = Professor heavily emphasized; high exam likelihood

### Export for Exam Day

Filter for ⭐⭐ and ⭐⭐⭐ sources → Create condensed reference sheet for `99_EXAM_DAY/`.
