## Strategic Usage

### Phase 1: During Semester

**Before each class:**
1. Read assignment from casebook
2. Check **2021 outline** for detailed case analysis
3. Review **2024 outline** for quick rule synthesis

**After each class:**
1. Update your personal notes
2. Cross-reference with **2024 outline** for Prof. Lee's emphasis
3. Use **2021 outline** to clarify any confusion

**For problem sets/hypotheticals:**
- Start with **2024 outline** for quick rule application
- Check **2021 outline** for edge cases and competing approaches

---

### Phase 2: Exam Prep

**First pass (3-4 weeks before exam):**
- Use **2024 outline** as primary study guide
- Reflects most recent teaching style
- Efficient coverage of all topics

**Second pass (2 weeks before exam):**
- Drill weak topics using **2021 outline** for depth
- Focus on areas where you need more examples

**Final review (week of exam):**
- **2024 outline only** for efficient review
- Create your own attack outline/checklist

**Exam day:**
- Bring **2024 outline** (shorter, easier to navigate under time pressure)
