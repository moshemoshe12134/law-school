## Usage Guidelines

### ✓ DO:
- Reference these materials when creating prep packets
- Extract key language for outline inserts
- Cite to page numbers in crosswalks
- Analyze past exams for exam spec

### ✗ DON'T:
- Edit or annotate PDFs in this directory
- Create "working versions" here
- Move files out (keep originals intact)

**Working Notes Go In**:
- `04_PREP/` - prep packets with extracted content
- `05_CLASS_NOTES/` - class-specific notes
- `07_OUTLINE/` - synthesized outlines
