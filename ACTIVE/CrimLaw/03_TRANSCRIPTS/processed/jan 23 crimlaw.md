
Common law, 4 types of mens rea; irrelevent for class or for life, important only for studying for the Bar. 
e.g., 'Malice' exists only in common law.

MPC:
4 types of mens rea:
- purposefully
- knowingly
- recklessly
- negligently

not all old cases fit squarely into these 4 types but we should seek to fit all cases into it.

3 types of element for a crime:
-  Conduct
- Attendant circumstance
- Result
