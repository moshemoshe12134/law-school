## Naming Conventions

- **Folders:** lowercase with underscores (`past_exams/`, `lecture_slides/`)
- **Exam folders:** `YYYY_Semester` (e.g., `2025_Spring`)
- **Assignment files:** `YYYY-MM-DD_classNN.md`
- **Slide folders:** `YYYY_Semester` (e.g., `2025_Spring`)

## Usage Notes

1. **Start here:** Review `syllabus/assignments/` for each class
2. **Exam prep:** Use `past_exams/` for practice - recent exams most relevant
3. **Lecture review:** Reference `lecture_slides/` for professor's visual materials
4. **TA sessions:** `ta_slides/2025_spring/` for current semester review
5. **Deep dives:** `readings/` for supplemental theory and policy

## Course-Specific Notes

**Professor Emphasis:**
- **MPC text** - Model Penal Code provisions central
- **Policy arguments** - ~50% of exam weight
- **Critical theory** - Kelman, Balkin, Foucault readings
- **Race and criminal law** - Steiner Bright emphasis on systemic issues

**Exam Pattern:**
- Issue-spotting with policy depth
- Hypothetical fact patterns
- Open-book (typically)
- Emphasis on MPC statutory analysis
