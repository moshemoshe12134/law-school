# 04_PREP (Outputs You Actually Use)

This folder holds the **two documents you use before/during class**.

For the system spec and per-class coverage checklist, see:
- `Admin/Two_Doc_Prep_System.md`
- `Admin/Per_Class_Prep_Blueprint.md`

## Folder Structure (active)
```
Lectures/Prep/
├── audio_preps/          ← TTS-friendly scripts you listen to before class
├── class_references/     ← Searchable Q&A references for use during class
├── AUDIO_PREP_STYLE_GUIDE.md
├── CLASS_REFERENCE_STYLE_GUIDE.md
└── README.md (this file)
```

## What to generate (per class)
1. Audio prep: `Lectures/Prep/Audio/YYYY-MM-DD_classNN_audio_prep.md`
2. Class reference: `Lectures/Prep/Reference/YYYY-MM-DD_classNN_class_reference.md`

Templates + QC live in:
- `Admin/Templates/Audio_Prep_TEMPLATE.md`
- `Admin/Templates/Class_Reference_TEMPLATE.md`
- `Admin/Templates/QC_Checklists.md`

## How to use (no reading required)
- **Before class:** listen to the audio prep (1.5x–2.0x), then skim the class reference’s Quick Navigation.
- **During class:** keep the class reference open and search it when called on.
- **After class (optional, 5 minutes):** jot down deltas and exam signals; later, fold them into your reference doc.

## Notes
- Older “prep packet” drafts live directly under `Lectures/Prep/` (e.g., `Lectures/Prep/Packets/2026-01-22_class02_prep.md`). They’re optional scaffolding and can be archived once the two primary docs are generated.
- Archived/deprecated items are tracked in `Admin/Archive.md`.
